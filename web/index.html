NFL Win Matrix

2010 Season

Order:

Each colored cell represents that the row team beat the column team. Darker cells indicate that they won twice.

Use the drop-down menu to reorder the matrix and explore the data.

Loss:

The loss function is the number of times a higher ranked team beat a lower ranked team.

This can be visualized as the number of colored squares beneath the diagonal.

See my blog post for information on ranking models.
